Video Game Sales Analysis

Author

Florijan Peric

Introduction

This project analyzes the Video Game Sales dataset. The goal is to investigate:

  • Which genres generate the most sales.
  • How video game sales changed over time.
  • Which publishers dominate the market.
  • Regional differences in sales.

Column Description

  • Rank: Sales ranking.
  • Name: Game title.
  • Platform: Gaming platform.
  • Year: Release year.
  • Genre: Game genre.
  • Publisher: Publisher.
  • NA_Sales: North America sales (millions).
  • EU_Sales: Europe sales (millions).
  • JP_Sales: Japan sales (millions).
  • Other_Sales: Other regions.
  • Global_Sales: Worldwide sales.

User Defined Function

top_n_sales <- function(data, n = 10) {
  data |>
    arrange(desc(Global_Sales)) |>
    slice_head(n = n)
}

Plot 4: Regional Sales

Code 
region_sales <- games |>
  summarise(
    North_America = sum(NA_Sales, na.rm = TRUE),
    Europe = sum(EU_Sales, na.rm = TRUE),
    Japan = sum(JP_Sales, na.rm = TRUE),
    Other = sum(Other_Sales, na.rm = TRUE)
  ) |>
  pivot_longer(
    everything(),
    names_to = "Region",
    values_to = "Sales"
  )

ggplot(region_sales,
       aes(x = Region, y = Sales)) +
  geom_col() +
  labs(
    title = "Regional Sales Comparison",
    x = "Region",
    y = "Sales"
  ) +
  theme_minimal()

Interpretation: North America contributes the largest share of sales.

Conclusion

The analysis shows that:

  1. Action and Sports are among the most successful genres.
  2. A small number of publishers account for a large portion of total sales.
  3. Global sales increased strongly before reaching a peak and later declining.
  4. North America is the largest market for video games.

The project demonstrates the use of tidyverse tools including filter(), mutate(), summarise(), inner_join(), ggplot(), and user-defined functions.
